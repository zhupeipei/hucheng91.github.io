BOM对象是浏览器对象模型的简称，像我们常用的window，location，等，这些在javascript中是没有定义的，浏览器为了使javacript在浏览器中运行，更方便的操作，就定义这类对像；由于没有规范，导致浏览器供应商会按照自己的想法扩展，W3C 为了把浏览器中JavaScript 最基本的部分标准化，已经将BOM的主要方面纳入了HTML5 的规范中。

window 对象

BOM 的核心对象是window，它表示浏览器的一个实例。在浏览器中，window 对象有双重角色，它既是通过JavaScript访问浏览器窗口的一个接口，又是ECMAScript 规定的Global对象。

##做Global对象用
意味着在网页中定义的任何一个对象、变量和函数，都以window 作为其Global 对象；定义的全局变量，函数其是都是绑定在window上，举个例子

##窗口位置

  • 浏览器据屏幕左侧距离 ：window.screenX/window.screenLeft

  • 浏览器据屏幕上侧距离 ：window.screenY/window.screenTop

    ##窗口大小

  • outerWidth、outerHeight

  • innerWidth、innerHeight

    ##打开窗口window.open(URL,name,features,replace)

  • URL 一个可选的字符串，声明了要在新窗口中显示的文档的 URL。如果省略了这个参数，或者它的值是空字符串，那么新窗口就不会显示任何文档。

  • name 一个可选的字符串，该字符串是一个由逗号分隔的特征列表，其中包括数字、字母和下划线，该字符声明了新窗口的名称。这个名称可以用作标记 和

    的属性 target 的值。如果该参数指定了一个已经存在的窗口，那么 open() 方法就不再创建一个新窗口，而只是返回对指定窗口的引用。在这种情况下，features 将被忽略。

  • features 一个可选的字符串，声明了新窗口要显示的标准浏览器的特征。如果省略该参数，新窗口将具有所有标准特征。具体参考地址
    replace
    一个可选的布尔值。规定了装载到窗口的 URL 是在窗口的浏览历史中创建一个新条目，还是替换浏览历史中的当前条目。支持下面的值：
    true - URL 替换浏览历史中的当前条目。
    false - URL 在浏览历史中创建新的条目。

setTimeout(para,time),setInterval(para,time)

setTimeout()超时调用使用注意事项

  • para尽量不要用字符串，用字符串会调用eval()函数，容易卡死，多用function{};如 setTimeout(function(){console.log(11)},100)

  • 使用后记得清除，调用clearTimeout(id),这样在时间到了清除

setInterval()间歇调用，同setTimeout()一样参数，意思是间隔多长时间执行一次，相当定时器
