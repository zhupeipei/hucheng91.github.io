data方法源码

$(element).data(key,data)，直传入2个参数，它地层调用data: function( elem, name, data )，将$(element)作为参数传入，data的数据缓存是给element新增一个属性elem[ expando ]，

$("#test1").data("data1",12);
$("#test1").data("data2",12);
$("#test2").data("data3",12);
//二个元素新增一个属性elem[ expando ] 是下面这样
$("#test1")[expando] = 2;
$("#test2")[expando] = 3;
//二个元素的elem[ expando ]对应cache存储是这样的
cache :{
        2 : {"data1"：12，"data2",12},
        3 : {"data3",12}
}
